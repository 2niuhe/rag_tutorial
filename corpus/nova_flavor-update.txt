# Command: nova help flavor-update

nova CLI is deprecated and will be removed in a future release
usage: nova flavor-update <flavor> <description>

Update the description of an existing flavor. (Supported by API versions
'2.55' - '2.latest') [hint: use '--os-compute-api-version' flag to show help
message for proper version]

Positional arguments:
  <flavor>       Name or ID of the flavor to update.
  <description>  A free form description of the flavor. Limited to 65535
                 characters in length. Only printable characters are allowed.
